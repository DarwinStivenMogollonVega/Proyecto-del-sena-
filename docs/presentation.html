Layout principal

Listado y detalle

Carrito y pedido

Panel de control (Admin)
Vista: Panel de control
  • Archivo: dashboard.blade.php
  • Tecnologías: Blade admin layout (`plantilla.app`), Bootstrap, Chart.js (CDN), CSS admin (`dashboard-section.css`).
  • Widgets: tarjetas de métricas, tablas responsivas, gráficos (Chart.js), barras de progreso visuales.
  • Funciones: `@can`/`@canany` para módulos, inclusión de módulos (Productos, Pedidos, Clientes, Facturas), llamadas a rutas administrativas (`admin.pedidos`, `admin.facturas`, `usuarios.index`).
Vistas administrativas adicionales
  • Archivos localizados: admin/pedidos/index.blade.php, admin/facturas/index.blade.php, creación/edición de facturas.
  • Funciones: listas CRUD, permisos, acciones en lote y enlaces a generación de PDFs.
Patrones comunes
  • Blade + Bootstrap como base UI.
  • Icons: Bootstrap Icons CDN; CSS modular por sección (header, nav, index, item, checkout, admin).
  • Interactividad: vanilla JS (fetch, event delegation), Chart.js en admin, IntersectionObserver para animaciones.
  • Formularios: `@csrf`, `old()`, `$errors`, rutas nombradas y validación server-side.
  • Acceso: `@auth`, `@can`, `@canany`, checks inline con `auth()->user()->can()`.
