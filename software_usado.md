
# Software Usado

Este proyecto utiliza las siguientes tecnologías y herramientas:

- **Laravel**: Framework PHP para desarrollo backend y API REST.
- **PHP**: Lenguaje principal del backend.
- **Mysql**: Sistema de gestión de bases de datos relacional.
- **Bootstrap 5**: Framework CSS para diseño responsivo y componentes UI.
- **AdminLTE**: Plantilla de panel de administración basada en Bootstrap.
- **JavaScript**: Lenguaje de programación para interactividad en el frontend.
- **Vite**: Herramienta de build y recarga rápida para recursos frontend.
- **Spatie Permissions**: Paquete Laravel para gestión de roles y permisos.
- **Composer**: Gestor de dependencias PHP.
- **PHPUnit**: Framework de testing para PHP.
- **Selenium**: Herramienta para pruebas automatizadas de interfaz.
- **Git**: Control de versiones del código fuente.

Cada herramienta cumple un rol específico en el desarrollo, despliegue y gestión del sistema, asegurando robustez, seguridad y escalabilidad.
